## Project AP1 - Program Konversi Bilangan Antar Basis

Jadi program ini adalah sebuah alat untuk mengonversi suatu bilangan—yaitu bilangan bebasis 2, 8, 10, atau 16—ke basis lain yang berbeda dari input awal.
Projek ini disusun oleh Kelompok 1 Kelas C untuk pemenuhan tugas kelompok mata kuliah Algoritma dan Pemrograman.
